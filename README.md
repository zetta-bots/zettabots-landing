# ZettaBots — Plataforma SaaS de IA para WhatsApp

Site e painel do cliente oficial da ZettaBots ([zettabots.ia.br](https://zettabots.ia.br)).
Nossa plataforma permite que empresas criem, conectem e gerenciem Agentes de IA autônomos integrados ao WhatsApp em menos de 5 minutos.

## 🚀 O que entregamos
- **Landing Page de Alta Conversão:** Otimizada para SEO e captura de leads.
- **Dashboard SaaS Premium:** Gestão de clientes, acompanhamento de leads em tempo real e monitoramento de chat.
- **Integração N8N & Evolution API:** Arquitetura robusta para processamento de linguagem natural (LLM), áudios e fluxos de vendas.
- **Faturamento Automatizado:** Assinaturas gerenciadas via Mercado Pago com liberação e bloqueio automático de instâncias.

## 🛠️ Stack Tecnológico
- **Frontend:** React 18 + Vite
- **Estilização:** CSS Vanilla Premium (Glassmorphism, Dark Mode)
- **Backend / Auth / DB:** Supabase (PostgreSQL + RLS)
- **Deploy / Serverless:** Vercel (Funções em `api/`)
